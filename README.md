# Baking App

This project is the third task of the Android Nanodegree Scholarship. Allows users to playback media, add and remove items to/from widget and so on.

This app is built with MVVM Design Pattern
- LiveData
- ViewModel
- Room
- Data Binding
- Dagger
- Retrofit
- Exo Player
- Widget
- Espresso Test
